# Kubernetes Cluster Operations at DigitalOcean

## Build
```
make build-image
```

## View in a Browser
```
make present
```

Open up a browser to `127.0.0.1:8000` on Linux or your Docker VM ip address on Mac or Windows (usually 192.168.99.100:8000)
